# Roadmap

## Launch Readiness
- Reference checklist: [Launch Readiness Checklist](launch-readiness.md)

## Prioritized Roadmap
1. **Stabilize core orchestration**
   - Harden task lifecycle handling in the orchestrator.
   - Ensure Redis Stack schemas and indexes are production-ready.
2. **Expand agent execution layer**
   - Implement core agent workers and routing policies.
   - Add observability hooks for agent performance.
3. **Complete document ingestion**
   - Finalize docling service integration.
   - Support multimodal OCR and embeddings pipeline.
4. **Ship dashboard MVP**
   - Deliver a monitoring UI for tasks, approvals, and memory.
   - Add role-based access and visibility controls.
5. **Scale automation workflows**
   - Publish reusable n8n workflow templates.
   - Add integration catalog for external tools.
