# Mother Harness

A project for capturing and managing comic book photos.

## Getting Started

This project is set up to help you take and organize photos of your comic book collection.

## Project Structure

```
mother-harness/
├── photos/          # Raw comic photos
├── processed/       # Processed/edited photos
└── README.md
```

## Usage

1. Add your comic photos to the `photos/` directory
2. Processed images will be stored in `processed/`
